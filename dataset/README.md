# Dataset Directory

Place your training images here, organised in sub-directories named after the
person they represent.

## Expected structure

```
dataset/
├── Alice/
│   ├── alice_01.jpg
│   ├── alice_02.jpg
│   └── alice_03.jpg
├── Bob/
│   ├── bob_01.jpg
│   └── bob_02.jpg
└── ...
```

## Tips

* Use **clear, front-facing** photos for best accuracy.
* Include **at least 5–10 images** per person taken under different lighting
  conditions and angles.
* Supported formats: `.jpg`, `.jpeg`, `.png`, `.bmp`.
* Image resolution should be at least **200 × 200 pixels**.
